Ephemeral PR Preview Environment Platform

Completed

A Kubernetes-based CI/CD platform on Azure Kubernetes Service (AKS) that spins up fully isolated, per-branch preview environments for every pull request — automatically torn down after merge. Infrastructure is provisioned via Bicep, pipelines managed through Azure DevOps, and Helm handles per-namespace deployments. Scheduled cluster start/stop cuts idle costs significantly.
